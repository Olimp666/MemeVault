# Тестовый код для ИИ-модели по расшифровке видео
Код запускает локальный Flask веб-сервер с ИИ-моделью, дает ей указанное видео и сгенерированные теги записывает в description.txt

В директории memes находятся видео для тестов (можно взять и свои).

## Запуск локального Flask веб-сервера с моделью

Из caption_service нужно выполнить следующие команды:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Генерация описания к видео

Для генерации описания к видео нужно в друом терманале из AI-video-description выполнить:

```bash
go run main.go <путь_к_видео>
```
