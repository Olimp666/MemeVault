# Тестовый код для ИИ-модели, извлекающей текст из картинки
Код запускает локальный Flask веб-сервер с ИИ-моделью, дает ей указанную картинку а текст, который модель распознала на картинке,записывает в description.txt

В директории memes находятся картинки для тестов (можно взять и свои).

## Запуск локального Flask веб-сервера с моделью

Из caption_service нужно выполнить следующие команды:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Извлечение текста с картинки

Для извлечения текста с картинки нужно в другом терманале из AI-image-text-extraction выполнить:

```bash
go run main.go <путь_к_картинке>
```
