# Тестовый код для ИИ-модели по расшифровке картинок
Код запускает локальный Flask веб-сервер с ИИ-моделью, дает ей указанную картинку и сгенерированные теги записывает в description.txt

В директории memes находятся картинки для тестов (можно взять и свои).

## Запуск локального Flask веб-сервера с моделью

Из caption_service нужно выполнить следующие команды:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Генерация описания к видео

Для генерации описания к картинке нужно в другом терманале из AI-image-description выполнить:

```bash
go run main.go <путь_к_картинке>
```
