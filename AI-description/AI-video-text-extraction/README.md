# Тестовый код для ИИ-модели, извлекающей текст из видео
Код запускает локальный Flask веб-сервер с ИИ-моделью, дает ей указанное видео а текст, который модель распознала на видео, записывает в description.txt

В директории memes находятся видео для тестов (можно взять и свои).

## Запуск локального Flask веб-сервера с моделью

Из caption_service нужно выполнить следующие команды:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

## Извлечение текста с видео

Для извлечения текста с видео нужно в другом терманале из AI-video-text-extraction выполнить:

```bash
go run main.go <путь_к_видео>
```
